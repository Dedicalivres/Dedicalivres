Dédicalivres est un agenda participatif édité par une association. Les informations ci-dessous précisent l’entité responsable du site et les services techniques utilisés.

Hébergement et services techniques

Les fichiers publics du site sont publiés via GitHub Pages.

GitHub, Inc. — 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, États-Unis.

Le site utilise également Supabase pour la base de données et l’authentification, ainsi que Cloudflare Workers et Cloudflare R2 pour certains services techniques comme les images, les exports ou les traitements automatisés.

Nature des informations publiées

Les événements, présences auteurs et témoignages publiés sur Dédicalivres sont issus de contributions participatives ou de recherches éditoriales. Ils sont relus avant publication.

Malgré le soin apporté à la vérification, les informations restent indicatives. Pour une information officielle et à jour, il est recommandé de consulter le site de l’événement, de l’organisateur, de la librairie ou de l’auteur concerné.
